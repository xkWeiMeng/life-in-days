# Living Days — 人生可视化进度

> 每天一个格子，看见你的人生进度。

## 项目结构

| 文件 | 用途 |
|------|------|
| `index.html` | 单文件应用（HTML + CSS + JS） |
| `specs/` | [规范目录](specs/README.md) — SDD 驱动 |

## 快速开始

直接在浏览器中打开 `index.html` 即可使用，无需任何构建工具或服务器。

## 规范

所有产品需求和设计决策记录在 `specs/` 目录中：

- [产品规范](specs/product-specs/) — 功能需求
- [设计文档](specs/design-docs/) — 技术方案
- [执行计划](specs/exec-plans/) — 任务分解
- [决策记录](specs/decisions/) — 架构决策 & Harness 反馈
