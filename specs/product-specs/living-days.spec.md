# Living Days — 人生可视化进度

## 概述

一个纯前端单文件网页应用，将用户的人生以天为单位可视化为格子矩阵。用户输入出生日期和预期寿命，即可看到已经度过的日子（霓虹绿格子）和未来的日子（深灰格子），以及底部的百分比进度。

## 能力

### CAP-1: 用户设置

用户可以输入出生日期和预期寿命。

**WHEN** 用户首次打开页面且 localStorage 无数据
**THEN** 显示设置面板，出生日期为空，预期寿命默认 82 年

**WHEN** 用户输入出生日期为 2000-01-01，预期寿命为 82
**THEN** 计算总天数 = 82 × 365.25 ≈ 29,930 天（考虑闰年）
**AND** 将设置保存到 localStorage

**WHEN** 用户修改预期寿命为 90
**THEN** 立即重新计算并刷新格子和进度

**WHEN** 用户输入无效日期（如未来日期）
**THEN** 显示错误提示，不保存

### CAP-2: 格子可视化

以格子矩阵呈现人生总天数。

**WHEN** 设置完成后
**THEN** 使用 Canvas 渲染格子矩阵
**AND** 已过天数格子为霓虹绿色（#00ff88）带发光效果
**AND** 未来天数格子为深灰色（#1a1a2e）
**AND** 今天的格子有特殊高亮标记

**WHEN** 屏幕宽度 ≥ 768px（桌面端）
**THEN** 格子尺寸适中，每行显示尽可能多的格子

**WHEN** 屏幕宽度 < 768px（移动端）
**THEN** 格子缩小，保持可视性

### CAP-3: 进度统计

在格子下方显示统计信息。

**WHEN** 设置完成后
**THEN** 显示已过天数 / 总天数（如 "9,321 / 29,930"）
**AND** 显示百分比（如 "31.14%"）
**AND** 显示霓虹风格进度条

**WHEN** 已过天数超过总天数（用户已超过预期寿命）
**THEN** 百分比显示为 100%+，进度条满格且特殊颜色提示

### CAP-4: 本地持久化

数据存储在浏览器 localStorage。

**WHEN** 用户设置出生日期和预期寿命后
**THEN** 数据以 JSON 格式存入 localStorage key "life-in-days-config"

**WHEN** 用户再次打开页面
**THEN** 自动从 localStorage 加载配置
**AND** 直接显示格子和进度，不再弹出设置面板

**WHEN** 用户点击"重新设置"按钮
**THEN** 显示设置面板，允许修改

### CAP-5: 双语支持

支持中文和英文切换。

**WHEN** 用户首次打开
**THEN** 根据浏览器语言自动选择中文或英文

**WHEN** 用户点击语言切换按钮
**THEN** 所有界面文本立即切换到目标语言
**AND** 语言偏好保存到 localStorage

## 非功能需求

- **性能**：~30,000 格子使用 Canvas 渲染，首次渲染 < 500ms
- **兼容性**：现代浏览器（Chrome、Firefox、Safari、Edge）
- **零依赖**：不使用任何第三方库
- **单文件**：所有代码在一个 index.html 中
